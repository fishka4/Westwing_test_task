To launch this solution Node js should be installed.
Chrome browser should also be up to date.

Please follow the steps below before test case launch.

1. Extract Westwing_test_task archive.
2. Open Westwing_test_task folder in Visual Studio code.
3. Execute 'npm install' command in VSCode terminal to install all required dependencies.
4. Execute 'npm run test' command in VSCode terminal to start test case execution.
